# 🎬 Movie & Series Recommendation System

A full-stack web application that provides personalized movie and TV series recommendations based on user preferences and personality.

---

## ✨ Overview

This project is a modern recommendation system designed to help users discover movies and TV series that match their interests, mood, and personality.

The application combines:
- a **personality-based recommendation system**
- a **manual filtering system**
- a **modern and animated UI**
- integration with a real-world API (TMDB)

---

## 🚀 Features

### 🔐 Authentication
- User registration
- User login & logout
- Session-based experience

### 🧠 Personality Test
- 15-question personality quiz
- Generates personalized recommendations
- Enhances recommendation accuracy

### 🎬 Movie Recommendations
- Filter by:
  - Genre
  - Mood
  - Duration
  - Language
- Smart scoring system

### 📺 TV Series Recommendations
- Same system as movies
- Unified experience

### ▶️ Trailer Integration
- Click on a movie/series → opens YouTube trailer

### ⭐ Watch & Rating System
- Mark movies/series as **watched**
- View personal watched list
- Give ratings to watched content
